What is?

A self-hosted server, which started with the idea of developing useful services, concentrated on a simple RaspberryPi.

The purpose was to feed my desire to learn full-stack web development and programming skills in general, as well as to familiarize myself with Linux devices, databases, etc.

Now the purpose is still the same, although it has evolved a lot since I started this website in October 2023.
